# 🧱 Node Design Rules

> **Node = Business Semantic Unit**

## ✅ Node must:

- Have clear input/output schema
- Be named with business meaning
- Be small and understandable

## 🔵 Node can:

- Throw Biz Error

## 🚫 Node must not:

- Control flow
- Access global state directly

## 🎯 Priority: Use framework utility first
