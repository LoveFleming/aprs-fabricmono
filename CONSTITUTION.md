# AI Software Factory 憲法

這份憲法定義了 AI Software Factory 的**六大核心原則**：

| # | 原則 | 核心精神 |
|---|------|----------|
| 1 | **Spec is Single Source of Truth** | 規格是所有工作的共同依據 |
| 2 | **No Spec, No Code** | 沒有規格，就不開始寫正式程式碼 |
| 3 | **Code must be generated or checked from Spec** | 程式碼必須能追溯回規格 |
| 4 | **Every Node must have contract** | 每個 Node 都必須有明確資料契約 |
| 5 | **Every API must have test and runbook** | 每條 API 都必須有測試與 Runbook |
| 6 | **AI can help, but gates decide** | AI 可協助，但最終由 Gate 把關 |

---

## 🏭 工廠宣言

> **"We do not build software by guessing."**
> **"We build software from Spec, through Gates, with AI as our accelerator."**
